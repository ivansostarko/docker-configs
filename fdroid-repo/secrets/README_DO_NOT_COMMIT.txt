Place your secret material here.

Required (if you sign indexes here):
- keystore.p12
- keystore.pass        (one line)
- keypass.pass         (one line)
- keyalias.txt         (one line; e.g. myrepo)

Optional (only if you use FDROID_GPGSIGN=1):
- repo-signing.key
- repo-signing.pass

Never commit this directory. Prefer using a proper secret manager in production.
